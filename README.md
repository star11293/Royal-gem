# Royal Gem Home Health Care LLC — Website

A modern, animated single-page website for Royal Gem Home Health Care LLC, a compassionate in-home care provider based in Pikesville, Maryland.

## Files

- `index.html` — Main page
- `style.css` — All styles & animations
- `main.js` — Interactivity & effects
- `hero-consultation.jpg` — Hero & gallery image
- `companion-care.jpg` — Gallery image
- `about-banner.png` — About section image
- `about-collage.png` — Gallery image
- `contact-nurse.png` — Contact section image

## Hosting

Deployed on Netlify. Connect this repo and it will auto-deploy from `index.html`.

## Tech Stack

HTML5 + CSS3 + Vanilla JavaScript — zero dependencies.
