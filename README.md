# Royal Gem Home Health Care LLC — Website

A modern, animated single-page website for Royal Gem Home Health Care LLC, a compassionate in-home care provider based in Pikesville, Maryland.

## Features

- **Page loader** with animated progress bar
- **Sticky navbar** with glassmorphism blur + mobile hamburger menu
- **Scroll-reveal animations** (fade up/left/right/scale) via IntersectionObserver
- **3D tilt effect** on cards (desktop)
- **Magnetic buttons** that follow your cursor
- **Parallax blobs** in the hero that react to mouse movement
- **Animated counters** that count up when scrolled into view
- **Infinite marquee** trust strip
- **Ripple click effects** on all buttons
- **Image parallax** on scroll
- **Active nav highlighting** based on scroll position
- **Contact form** with loading spinner + success state
- **Newsletter subscribe** with confirmation feedback
- **Back-to-top button** with scroll visibility toggle
- **Fully responsive** — mobile-first breakpoints at 480px, 768px, 1024px

## Setup

No build tools needed — just open `index.html` in a browser or deploy to any static host (GitHub Pages, Netlify, Vercel, etc.).

### GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/repo-name`

## Tech Stack
